# OCR Food

OCR Food is a React + Vite app for meal OCR, nutrition analysis, and recommendation workflows.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm run test
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase
